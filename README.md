ApneApp: Proyecto de Breathwork y Apnea
¡Bienvenido/a al repositorio de ApneApp! Este espacio centraliza el desarrollo de dos versiones distintas de una misma aplicación web diseñada para explorar y entrenar las técnicas de respiración (Breathwork) y apnea (retención de la respiración).

En este repositorio encontrarás dos caminos diferentes para mejorar tu capacidad respiratoria, cada uno con un enfoque y una estética propios.

Versiones de la Aplicación
Ofrecemos dos experiencias complementarias:

1. Versión: Guía Integral de Breathwork
(Basada en la interfaz mostrada arriba)

Esta versión ofrece una plataforma centralizada para practicar diversas disciplinas de respiración. Con una interfaz limpia y moderna en modo oscuro, te guía a través de técnicas fundamentales para el bienestar físico y mental.

Secciones Incluidas:

🫁 Apnea: Ejercicios como "Apnea Simple" y las variantes de "Cruzada Ascendente/Descendente" con indicaciones claras sobre el llenado pulmonar (Lleno 100%, 70%, 30%).

🏳️ Respiración Guiada: Incluye sesiones populares como el método "Wim Hof" con rondas automáticas y retención, y prácticas de "Pranayama" con ratios configurables (Box, 4-7-8, Nadi Shodhana).

🧘 Meditación: Temporizador para prácticas como Anapana, Vipassana y Metta, con gongs configurables.

2. Versión: Entrenamiento de Apnea Estática
(La segunda versión en tu repositorio)

Esta versión se enfoca exclusivamente en la apnea estática, diseñada específicamente para nadadores, apneístas y cualquier persona interesada en mejorar su capacidad de retención y control del reflejo de inmersión. Cuenta con un diseño dedicado para sesiones de entrenamiento puro.

Características:

Enfoque Específico: Temporizador y herramientas optimizadas para apnea estática (retención en reposo).

Seguimiento del Entrenamiento: Facilita la práctica estructurada y el registro de tiempos para monitorizar el progreso.

Interfaz Funcional: Diseñada para la concentración y la claridad durante las sesiones.

# Breathwork

ApneApp: Breathwork & Apnea Project
Welcome to the ApneApp repository! This project hosts two distinct versions of a web application designed to help users train and master breathing techniques (Breathwork) and breath-holding (Apnea).

This repository serves as a hub for two different training paths, each with its own specific focus and UI design.

Application Versions
We offer two complementary experiences:

1. Version: Comprehensive Breathwork Guide
(Based on the dashboard interface)

A centralized platform for practicing various breathing disciplines. With a sleek, modern dark-mode interface, it guides you through fundamental techniques for physical and mental well-being.

Included Sections:

🫁 Apnea: Exercises like "Simple Apnea" and "Ascending/Descending Cross" variants with specific lung capacity instructions (Full 100%, 70%, 30%).

🏳️ Guided Breathing: Popular sessions including the "Wim Hof" method with automatic rounds and retention, and "Pranayama" with configurable ratios (Box, 4-7-8, Nadi Shodhana).

🧘 Meditation: A dedicated timer for Anapana, Vipassana, and Metta practices, featuring configurable gongs.

2. Version: Static Apnea Trainer
(The second version in this repository)

This version is laser-focused on static apnea, specifically designed for swimmers, freedivers, and anyone looking to increase their CO2 tolerance and dive reflex control.

Features:

Specific Focus: Optimized timers and tools for static apnea (breath-holding at rest).

Training Logic: Facilitates structured practice sessions to help monitor progress over time.

Functional Interface: Stripped-back design to ensure maximum focus during intense holds.

How to Access the Apps
Both versions are available to use live via GitHub Pages. Explore them and find the one that best fits your training goals!
